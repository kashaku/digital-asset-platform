# 项目启动指南

## 📋 项目概述

这是一个基于区块链的数字资产平台，包含：

- **后端**：Hardhat + Solidity 智能合约（Polygon Amoy / Sepolia / 本地节点）
- **前端**：React 19 + Vite + ethers.js（Web3 直连）

---

## 🚀 快速启动（3 步）

### 第一步：安装所有依赖

```bash
npm run install:all
```

### 第二步：部署智能合约

**选择 A - 本地 Hardhat 节点（推荐）**

终端 1 - 启动本地节点：

```bash
npm run node
```

终端 2 - 部署合约：

```bash
cd contracts
npx hardhat run scripts/deploy.ts --network localhost
```

记录输出的合约地址，如：

```
✅ DigitalAssetNFT 已部署到: 0x5FbDB2315678afccb333f8a9c36f1f3d...
```

**或选择 B - 部署到 Polygon Amoy 测试网**

创建 `.env` 文件：

```bash
cd contracts
cat > .env << EOF
PRIVATE_KEY=你的_私钥
POLYGON_AMOY_RPC_URL=https://rpc-amoy.polygon.technology/
EOF
```

部署：

```bash
npx hardhat run scripts/deploy.ts --network polygonAmoy
```

### 第三步：配置合约地址

编辑 `frontend/src/config/contract.ts`：

```typescript
export const CONTRACT_ADDRESS = "0x..."; // 替换为部署输出的地址
```

### 第四步：启动前端

```bash
npm run dev
```

浏览器打开：`http://localhost:5173`

## 🔌 测试前后端通信

### 1. 连接钱包

在前端页面右上角点击 **"连接钱包"** 按钮，授权 MetaMask 连接

### 2. 浏览器 DevTools 测试

打开 DevTools（F12）→ Console 标签页，执行以下命令：

**查询 NFT 总数：**

```javascript
const { useNFTContract } = await import("/src/hooks/useNFTContract.ts");
const contract = useNFTContract();
const total = await contract.getTotalMinted();
console.log("📊 总 NFT 数量:", total.toString());
```

**查看钱包连接状态：**

```javascript
const { useWalletStore } = await import("/src/store/wallet.ts");
const wallet = useWalletStore();
console.log("💰 钱包地址:", wallet.address);
console.log("🔗 Chain ID:", wallet.chainId);
console.log("✅ Provider:", wallet.provider ? "已连接" : "未连接");
```

**铸造测试 NFT：**

```javascript
const contract = useNFTContract();
const wallet = useWalletStore();

try {
  const receipt = await contract.mintAsset(
    wallet.address,
    "ipfs://QmTest",
    500,
  );
  console.log("✅ 铸造成功！Tx:", receipt.transactionHash);
} catch (error) {
  console.error("❌ 失败:", error.message);
}
```

## ⚠️ 常见问题

### "请先连接钱包"

- 点击页面右上角的钱包按钮连接
- 确保已安装 MetaMask 浏览器插件

### "CONTRACT_ADDRESS 是 0x0000..."

- 在 `frontend/src/config/contract.ts` 中更新为实际部署的合约地址

### "合约调用失败"

- 检查钱包是否在正确的网络上（本地节点、Polygon Amoy 等）
- 本地节点需要先启动 `npm run node`
- 确保 provider 已连接：`wallet.provider !== null`

### "交易提交后报错"

- 检查钱包中是否有足够的 ETH（本地测试可忽略）
- 查看浏览器 Console 中的完整错误信息

## 💡 开发提示

- **修改合约后**：运行 `npm run compile` 更新 ABI
- **查看合约日志**：在 hardhat.config.ts 中调整日志级别
- **本地测试**：使用 `npm run node` 提供的测试账户（已预充 ETH）
